Git Comandos basicos

Comandos iniciais de git bash

    • Comandos de login

    • git config --global user.name "nome do usuario do site do github" → comando que indica para o git bash qual o usuario que esta logando.
    • git config -- global user.email email@gmail.com → comando que indica qual o email que esta logando.

    • Comandos de logout

    • git config --global --unset user.name "nome do usuario do site do github" → Comando remove usuario atual do git hub.
    • git config --global --unset user. email. → Comando que remove o email utilizado do github
