Git Comandos basicos

Comandos iniciais de git bash

  • Comandos de login

    • git config --global user.name "nome do usuario do site do github" → comando que indica para o git bash qual o usuario que esta logando.
    • git config -- global user.email email@gmail.com → comando que indica qual o email que esta logando.

  • Comandos de logout

    • git config --global --unset user.name "nome do usuario do site do github" → Comando remove usuario atual do git hub.
    • git config --global --unset user. email. → Comando que remove o email utilizado do github

  • comandos de rastereio de pasta

    • git init → antes de dar esse comando esteja na pasta que deseja colocar para o servidor

  • rastrear versões do arquivo

    • git add "nome-do-arquivo.estenção" → coloca arquivo em rastreamento.
    • git rm --cached -r "nome-do-arquivo.estenção" → Remove arquivo do rastreamento de versão.

  • enviando versão para update " Commit"

    • git commit -m "mensagem de identificação do que foi feino no arquivo."
    • git log nome-do-arquivo.estenção → indica todas as versões , datas e autor da auteração do arquivo.

  • Clonando

    • git clone github.com/gabrielamorim/Java_repository.git → clona o repositorio para a minha branc.
    • git push → envia todos os commits do repositorio local para o servidor.
    • git pull → recebe atualização dos arquivos do servidor.

  • Branch ou Novos caminhos

    • git branch → indica os caminhos que um codico esta tomando
    • git branch nome-da-versão → cria um novo caminho de versão do arquivo.
    • git checkout nome-da-versão → muda para o caminho daquela versão do arquivo.
    • git checkout -b nome-da-versão → cria um caminho novo de versão e muda diretamente para ele.
    • git tag nome-da-tag identificador-do-commit → mescla o caminha diferente com o caminho principal.
